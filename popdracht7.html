De opdracht

Bij deze opdracht moesten we een formulier maken voor een website. We konden kiezen uit een taartformulier, een formulier voor een volkstuintje en voor het maken van een strip. Daarnaast moesten wij ervoor zorgen dat er slimme defaults waren verwerkt in het ontwerp. Ik heb voor het taartformulier gekozen. Ik vond het maken van het formulier erg lastig om te doen. Wat ik vooral moeilijk vond was het maken van creatieve input controls. Ik bleef erg hangen in de standaard input controls. Ik was dan ook niet heel blij met het uiteindelijke ontwerp.
